Tentang Saya

"Saya Amos, saya seorang mahasiswa dengan semangat dan ketertarikan yang besar dalam pengembangan web. Awalnya, saya menyelesaikan pendidikan di Sekolah Menengah Atas (SMA), tetapi kemudian memutuskan untuk melanjutkan studi di luar daerah asal saya. Saat ini, saya sedang menempuh pendidikan di salah satu perguruan tinggi dengan fokus pada bidang Informatika. Perjalanan saya di dunia teknologi dimulai setelah menyelesaikan pendidikan SMA. Keinginan saya untuk terus belajar dan mengikuti perkembangan teknologi terkini membawa saya lebih dalam ke dunia pengembangan web. Meskipun awalnya hanya seorang pelajar SMA, semangat belajar saya membawa saya ke tingkat pendidikan yang lebih tinggi dan merambah ke dunia yang saya minati, yaitu pengembangan web. Saya percaya bahwa pengembangan web adalah panggilan hati saya, dan saya berkomitmen untuk terus mengasah keterampilan dan pengetahuan saya di bidang ini. Setiap tantangan adalah peluang untuk tumbuh, dan saya sangat antusias menjalani perjalanan ini sebagai seorang mahasiswa di bidang Teknik Informatika dengan fokus pada pengembangan web."
